# FAP2026 — Formação Acelerada de Programação

Repositório e documentação pessoal do curso **FAP** (*Formação Acelerada de Programação*), com foco em **DevOps** e desenvolvimento de software.

---

## Sobre o FAP

**FAP** significa **Formação Acelerada de Programação** — um programa de capacitação intensiva voltado à formação de profissionais de tecnologia, com ênfase em práticas modernas de desenvolvimento, infraestrutura e entrega de software.

Este repositório (`FAP2026`) consolida informações pessoais para atividade academica da trilha sobre perfil profissional, stack tecnológica, objetivos de carreira entre outros...
